Это программа для автоматического построения Markdown-wiki и RAG-поиска по Pascal/Delphi-кодовой базе.

Она разработана, чтобы упростить анализ и сопровождение старых или плохо документированных Pascal/Delphi-проектов. Программа извлекает структуру исходного кода, формирует человекочитаемую wiki для Obsidian и одновременно готовит данные для AI-агента в OpenCode.

Основной сценарий работы:
Pascal/Delphi исходники
→ извлечение структуры через PasDoc
→ генерация Markdown-wiki
→ AI-обогащение описаний
→ индексация wiki в векторную базу
→ поиск по проекту через OpenCode

Программа позволяет:
- находить unit, классы, методы и связи между ними;
- получать краткие описания сущностей проекта;
- использовать Obsidian как документационную wiki;
- использовать OpenCode как AI-агента, который ищет контекст по проекту через RAG;
- задавать вопросы по кодовой базе на естественном языке;
- быстрее ориентироваться в проекте без ручного просмотра всех исходников.

Итоговая цель разработки — создать локальный инструмент, который связывает исходный Pascal/Delphi-код, автоматически созданную документацию и AI-агента в единую рабочую систему для анализа проекта.
